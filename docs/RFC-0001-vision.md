# RFC-0001 Vision

## Mission
SnoutDB is a tiny embedded columnar database that transforms messy raw data into useful metrics and insights.

## Core Principles
- Single-file storage
- Embedded first
- Raw-data friendly
- Merge-friendly
- Insight-oriented

## Product Statement
Feed SnoutDB logs, events, metrics, and telemetry. It detects structure, extracts signals, and consolidates information into valuable analytics.
